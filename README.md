# Broccoli CSSLint

Add CSS linting to your [Broccoli](https://github.com/broccolijs/broccoli) tree.

## Installation

```
npm i broccoli-csslint
```

## Usage

```javascript
var cssLint = require('broccoli-csslint');

var tree = cssLint(someTree);
```

## Development

```
npm i
npm t
```
